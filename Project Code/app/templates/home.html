Know the flood risk
before the water rises.

Rising Waters reads rainfall, humidity, temperature and visibility like a station instrument panel, then turns those readings into a flood-risk call in under a second — powered by a machine learning model trained on historical weather patterns.

Run a prediction → How it works
01 / READ

Weather intake

Annual & seasonal rainfall, cloud visibility, temperature and humidity are collected as the model's input features.

02 / SCALE

Standardize

Inputs are transformed with the same fitted StandardScaler used during training, so readings stay consistent.

03 / PREDICT

XGBoost model

A gradient-boosted classifier, selected after comparison against Decision Tree, Random Forest and KNN, scores the risk.

04 / ACT

Clear call

You're routed to a flood-risk or all-clear readout, with practical next steps either way.
